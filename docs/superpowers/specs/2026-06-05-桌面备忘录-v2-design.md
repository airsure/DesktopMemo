# 桌面备忘录 v2 设计方案

日期：2026-06-05

## 目标

将现有单文件桌面备忘录重构为基于 PyQt5 的桌面应用，解决以下问题：
- 编辑不便（需要手动编辑桌面 txt 文件）
- 视觉效果简陋（文字直接画在背景图上）
- 背景图片兼容性差（只能用深色右侧背景）

## 架构

三个 PyQt5 窗口 + JSON 数据文件：

| 窗口 | 职责 | 关键属性 |
|------|------|---------|
| 桌面显示面板 | 桌面右侧展示任务列表，只读 | 鼠标穿透、置顶、无边框、半透明背景 |
| 编辑主界面 | 表格编辑、拖拽排序、分类管理 | 正常窗口，从托盘唤醒/隐藏 |
| 系统托盘 | 管理两个窗口的生命周期 | 右键菜单、快捷键 |

数据存储为 `~/备忘.json`，编辑界面修改 → 写入 JSON → 信号通知 → 显示面板刷新。

## 桌面显示面板

- 风格：圆点标记（彩色圆点替代符号前缀）
- 位置：桌面右侧
- 属性：`Qt.WindowStaysOnTopHint | Qt.FramelessWindowHint | Qt.Tool | Qt.WA_TransparentForMouseEvents`
- 背景：半透明，跟随主题
- 仅展示未被勾选完成的任务，按 sort 排序
- 已完成任务默认隐藏

## 编辑主界面

- 风格：统一列表，所有分类在同一视图
- 布局：顶部输入栏（文本输入框 + 分类下拉 + 添加按钮），下方任务列表
- 每行元素：☰ 拖拽手柄 → 分类圆点 → 复选框 → 任务文本
- 已完成的文本显示删除线 + 降低透明度
- 窗口标题：「工作备忘录」
- 关闭行为：最小化到托盘，不退出程序

## 配色主题

三套主题，通过托盘菜单切换：

| 主题 | 显示面板 | 编辑界面 |
|------|---------|---------|
| 暗夜 | 深色半透明底 `rgba(20,20,35,0.88)` | 深灰底 `#2b2b2b` |
| 明亮 | 白色底 `rgba(255,255,255,0.9)` | 浅灰底 `#f5f5f5` |
| 毛玻璃 | 模糊半透明 + 文字阴影 | 亚克力效果 |

五类任务的色彩保持一致：
- 进行中：红色 `#ff4757`
- 紧急：橙色 `#ff6348`
- 跟进：蓝色 `#1e90ff`
- 已完成：绿色 `#2ed573`
- 备忘：粉色 `#ff6b81`

## 数据模型

JSON 文件 `~/备忘.json`：

```json
{
  "version": 1,
  "theme": "dark",
  "tasks": [
    {
      "id": "a1b2c3",
      "text": "完成Q4报告",
      "category": "now",
      "done": false,
      "sort": 0
    }
  ]
}
```

字段：`id`（唯一标识）、`text`（任务文本）、`category`（`now`/`fire`/`follow`/`done`/`memo`）、`done`（完成状态）、`sort`（排序序号）。

## 交互行为

| 操作 | 方式 |
|------|------|
| 唤醒编辑界面 | 双击托盘图标，或托盘菜单 →「显示编辑」 |
| 隐藏编辑界面 | 关闭按钮 → 最小化到托盘，或 Esc |
| 添加任务 | 输入文本 → 选分类 → 回车或点 + |
| 切换完成状态 | 点击复选框 |
| 拖拽排序 | 拖拽 ☰ 手柄 |
| 切换分类 | 右键任务 → 选择新分类 |
| 编辑文字 | 双击任务文本进入编辑 |
| 删除任务 | 右键 → 删除，或选中后 Delete |
| 切换主题 | 托盘菜单 → 主题 → 暗夜/明亮/毛玻璃 |
| 退出程序 | 托盘菜单 → 退出，或 Ctrl+Alt+Q |

## 技术栈

- Python 3.x
- PyQt5（窗口管理、拖拽、托盘）
- JSON 文件存储
- 不再依赖 PIL/Pillow（不再生成壁纸图片）

## 文件结构

```
桌面备忘录/
├── 桌面备忘.py          # 旧版（保留不删）
├── main.py              # 新入口
├── app.py               # 应用主控（托盘、窗口生命周期）
├── overlay.py           # 桌面显示面板
├── editor.py            # 编辑主界面
├── data.py              # 数据读写层
├── theme.py             # 主题定义
├── requirements.txt     # PyQt5
└── 备忘.json            # 运行时数据
```
